# 15-Days of Advanced Deep Learning: Cheat Sheet

Welcome to your advanced Deep Learning codebase! Since this project is being pushed to your GitHub automatically, here is exactly what every script does so you can explain it in an interview.

**No API Keys:** This entire project is built from scratch using pure math (NumPy) and offline libraries (TensorFlow/PyTorch/OpenCV). There are NO API keys here, so your Secret Scanner will never block it!

## Part 1: Math & Fundamentals (Days 1-5)
We build the core math that powers all AI.
- **Day 1:** Linear Algebra (Matrix multiplication, eigenvectors).
- **Day 2:** Calculus (Derivatives, chain rule for backprop).
- **Day 3:** Forward Propagation (Passing data through neurons).
- **Day 4:** Loss Functions (MSE, Cross-Entropy).
- **Day 5:** Gradient Descent & Backpropagation (Updating weights to minimize loss).

## Part 2: Computer Vision (Days 6-10)
We teach the AI how to see images.
- **Day 6:** OpenCV basics (Edge detection, blurring).
- **Day 7:** Convolutions (Building CNN filters from scratch).
- **Day 8:** Pooling Layers (Max pooling, Average pooling).
- **Day 9:** Complete CNN Architecture (Putting it all together).
- **Day 10:** Regularization (Dropout, Early Stopping to prevent overfitting).

## Part 3: Natural Language Processing (Days 11-15)
We teach the AI how to read text (The foundation of LLMs like ChatGPT).
- **Day 11:** Tokenization & Bag of Words.
- **Day 12:** TF-IDF & Word Embeddings (Word2Vec concepts).
- **Day 13:** RNNs & LSTMs (Processing sequential text).
- **Day 14:** Self-Attention Mechanism (The core of Transformers).
- **Day 15:** Transformer Block Architecture (The final masterpiece).
